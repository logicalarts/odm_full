OpenDocMan is a full featured Web-based document management system designed to conform to ISO 17025/IEC. It features automatic installation, file check-in/out, departmental access control, file moderation, fine grained user access control, email notificaiton and a great search function. Written in PHP, and utilizing MySQL for the backend, this project is useful for any company looking to keep their documentation in a centralized repository.
